About

This website reports the check status of each package on R-multiverse.

Production

The Production status page lists the packages staged for the targeted Production snapshot. Packages are staged for Production if they pass R-multiverse checks at least once in the month-long period prior to the snapshot date.

Status summaries

The following pages list packages with R-multiverse check issues.

  • Community issues
  • Staging issues

Status details

Each package has its own human-readable status page and RSS feed for each of Community and Staging. For example, the multiverse.internals package has:

  • Community: status page, RSS feed
  • Staging: status page, RSS feed

Replace multiverse.internals with a different package name in any of the URLs above to navigate to the analogous content for that package. The package name part of each URL is case-sensitive.
